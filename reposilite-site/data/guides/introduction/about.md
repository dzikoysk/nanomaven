Reposilite *(formerly NanoMaven)* is a lightweight repository manager for Maven artifacts. 
It is a simple solution to replace managers like Nexus, Archiva or Artifactory.

Current site refers only to Reposilite 3.x.
If you're looking for Reposilite 2.x documentation, take a look at:

* [Reposilite 2.x](https://v2.reposilite.com)

### Platform

Reposilite is an open source project and its development takes place on GitHub:

* [GitHub / Reposilite](https://github.com/dzikoysk/reposilite)
